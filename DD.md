1. Функциональные детали

  Группы пользователей - пользователи телеграмма
  
  Функционал - набор данных о погоде в конкретном городе по запросу
  

2. Структура проекта

  Модули: Телеграмм-бот

  Интерфейс взаимодействия: Консоль

3. План выполнения
  Этапы разработки
    1) Создание проекта с базовой настройкой Maven
    2) Регистрация бота
    3) Реализация базового функционала бота (ответ на сообщения)
    4) Подключение к функционалу внешних источников информации
    5) Десериализация данных
    6) Конечная реализация функции отображения базовых параметров погоды

  Артефакт разработки : Телеграмм-бот, который может отвечать пользователю,  предоставляя информацию о погоде в названном городе
